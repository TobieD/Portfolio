This game is currently in development. It is a minimalistic game that requires the player to remember a pattern and recreate it in the correct order within a set amount of time. The pattern is shown for a small time at the beginning of each round. When the player succefully completes patterns the difficulty will increase.

This could be less time to solve the puzzle or different block that require a different approach

Technical Details

The game is being developed in Unity. I wanted to have the art style as simple as possible by only making use of blocks and colors. It also makes heavy use of Coroutines to handle the animations and color transitions.

To create the pattern I simply use a data table of integer values (currently 0-3), this system also allows me to easily add new types of blocks to the patterns.

Each value is a different type of block:

  • 0: Not part of the pattern.
  • 1: Part of the pattern.
  • 2: Start of the pattern.
  • 3: End of the pattern.
